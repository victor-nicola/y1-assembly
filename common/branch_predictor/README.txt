This .zip file should come with 4 files:
- Makefile
- example_test.a
- solution.s
- this file

Develop your solution as explained in the assignment manual inside the file "solution.s". To test it you can run `make test`, at that point a new file "out" will be outputted. Just run this with `./out` and you will know if your solution is ready for submit ;).

Please keep all the files in the same folder when compiling.
